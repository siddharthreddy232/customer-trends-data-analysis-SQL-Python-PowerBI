# 👨🏻‍💻 Retail Customer Behavior & Shopping Trends Analysis

This project demonstrates a complete end-to-end data analytics workflow using Python, PostgreSQL, and Power BI. The objective was to analyze customer shopping behavior, identify purchasing patterns, segment customers, and generate actionable business insights through data-driven analysis and visualization.

The project simulates a real-world business analytics scenario where customer transaction data is transformed into meaningful insights that support strategic decision-making.

---

# 📌 Project Overview

The primary goal of this project was to analyze retail customer behavior and shopping trends by leveraging data analytics tools and techniques.

The project includes:

✅ Data Cleaning and Exploratory Data Analysis using Python

✅ Database Integration and Business Analysis using PostgreSQL

✅ Interactive Dashboard Development using Power BI

✅ Business Reporting and Recommendation Generation

Through this project, I developed practical experience in data preparation, SQL analysis, dashboard design, and business storytelling.

# 📂 Dataset Information

The dataset contains customer shopping transactions and includes:

* Customer ID
* Age
* Gender
* Location
* Subscription Status
* Item Purchased
* Product Category
* Purchase Amount (USD)
* Shipping Type
* Previous Purchases
* Review Rating
* Discount Applied
* Promo Code Used
* Frequency of Purchases
* Season
* Color
* Size

### Dataset Summary

* Total Records: 3,900
* Total Features: 18
* Missing Values: Review Rating column contained missing values that were handled during preprocessing.

# 🐍 Data Preparation & Analysis Using Python

The first phase focused on preparing the dataset for analysis.

### Tasks Performed

### Data Loading

* Imported dataset using Pandas.

### Data Cleaning

* Identified missing values.

### Feature Engineering

Created:

### Exploratory Data Analysis

Analyzed:

* Customer demographics
* Purchase distributions
* Product preferences
* Revenue patterns

### Database Integration

Connected Python to PostgreSQL using SQLAlchemy and loaded the cleaned dataset into the database for further analysis.

# 🗄️ Business Analysis Using PostgreSQL

After loading the cleaned data into PostgreSQL, I answered several business questions using SQL.

### Key Business Questions

### 1. Revenue by Gender

Compared total revenue generated by male and female customers.

### 2. High-Spending Discount Users

Identified customers who used discounts but still spent above the average purchase amount.

### 3. Top Rated Products

Determined the products with the highest average review ratings.

### 4. Shipping Type Analysis

Compared average purchase amounts between Standard and Express shipping.

### 5. Subscription Analysis

Compared:

* Average Spending
* Customer Count
* Total Revenue

between subscribers and non-subscribers.

### 6. Discount-Dependent Products

Identified products with the highest percentage of discounted purchases.

### 7. Customer Segmentation

Segmented customers into:

* New Customers
* Returning Customers
* Loyal Customers

based on purchase history.

### 8. Top Products by Category

Ranked the most purchased products within each category.

### 9. Repeat Buyer Analysis

Investigated whether repeat buyers were more likely to subscribe.

### 10. Revenue Contribution by Age Group

Measured the contribution of different age groups toward total revenue.

# 💡 Key Insights

### Customer Behavior

* Most customers are non-subscribers.
* Loyal customers account for a significant portion of transactions.
* Repeat buyers show higher engagement levels.

### Product Performance

* Clothing generated the highest revenue.
* Certain products consistently received higher customer ratings.
* Some products depended heavily on discounts for sales.

### Revenue Analysis

* Revenue distribution varies across customer age groups.
* Subscription status does not significantly increase average purchase value but affects customer retention.

### Shipping Trends

* Express shipping customers demonstrated slightly higher spending behavior than Standard shipping users.
